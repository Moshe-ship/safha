Cleaning
10 steps. Zero noise.
Every scraped text runs through a multi-step cleaning pipeline before it touches your training data.
01

URL removal

Strip embedded links and raw URLs from text.

02

Email removal

Remove email addresses scattered in page content.

03

@mention removal

Strip social media handles and @mentions.

04

Alef normalization

Unify alef variants (أ إ آ) to bare alef (ا).

05

Ya normalization

Unify ya and alef maqsura (ي ← ى).

06

Tatweel removal

Strip kashida elongation characters (ـ).

07

Tashkeel stripping

Remove diacritics. Optional — use --keep-tashkeel.

08

Emoji removal

Strip emoji and special Unicode symbols.

09

Sentence splitting

Segment text into clean, individual sentences.

10

Quality filtering

Drop short, long, or low-Arabic-ratio text.

In Action
One command. Training-ready data.
Commands
6 commands. Full pipeline.

scrape

Scrape Arabic content from a URL. Extracts text, strips HTML, outputs structured JSONL.

sitemap

Crawl all URLs in a sitemap. Parallel fetching with rate limiting.

clean

Run the full cleaning pipeline on raw scraped data. 10 steps, configurable.

detect

Classify each text by Arabic dialect — MSA, Egyptian, Gulf, Levantine, Moroccan.

stats

Show dataset statistics — word count, dialect distribution, quality scores.

explain

Explain what each cleaning step does with before/after examples.

Get Started
Four commands to training data.
# Install $ pip install safha # Scrape a Wikipedia article $ safha scrape https://ar.wikipedia.org/wiki/... --output data.jsonl # Crawl an entire sitemap $ safha sitemap https://example.com/sitemap.xml # Clean the scraped data $ safha clean raw_data.jsonl --output clean.jsonl # Detect dialects $ safha detect data.jsonl # Show dataset stats $ safha stats clean.jsonl
