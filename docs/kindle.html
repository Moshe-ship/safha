Chapter I
The Pipeline
From raw web to clean training data.
Scrape → Clean → Detect → Export

Building Arabic AI models requires clean Arabic text data. Scraping the web gives you noise — HTML tags, ads, mixed languages, broken encoding. safha handles the full pipeline: scrape, clean, detect dialect, export. No API keys. No config.

Chapter II
Cleaning Pipeline
10 steps. Zero noise.
  1. URL removal
    Strip embedded links and raw URLs
  2. Email removal
    Remove email addresses
  3. @mention removal
    Strip social media handles
  4. Alef normalization
    Unify alef variants (أ إ آ) to bare alef (ا)
  5. Ya normalization
    Unify ya and alef maqsura (ي ← ى)
  6. Tatweel removal
    Strip kashida elongation (ـ)
  7. Tashkeel stripping
    Remove diacritics (optional: --keep-tashkeel)
  8. Emoji removal
    Strip emoji and special Unicode symbols
  9. Sentence splitting
    Segment into clean sentences
  10. Quality filtering
    Drop short, long, or low-Arabic-ratio text
Chapter III
Commands
6 commands. Full pipeline.
  • scrape
    Scrape Arabic content from a URL into JSONL
  • sitemap
    Crawl all URLs in a sitemap with rate limiting
  • clean
    Run the 10-step cleaning pipeline
  • detect
    Classify text by Arabic dialect
  • stats
    Show word count, dialect distribution, quality scores
  • explain
    Show before/after examples for each cleaning step

Chapter V
Get Started
# Install $ pip install safha # Scrape a Wikipedia article $ safha scrape https://ar.wikipedia.org/wiki/... --output data.jsonl # Crawl an entire sitemap $ safha sitemap https://example.com/sitemap.xml # Clean the scraped data $ safha clean raw_data.jsonl --output clean.jsonl # Detect dialects $ safha detect data.jsonl # Show dataset stats $ safha stats clean.jsonl
